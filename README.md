# SyslogMultilineLog

Tags each line in a multiline log with numbers

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'syslog_multiline_log'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install syslog_multiline_log
